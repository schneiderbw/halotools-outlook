Register a Halo Connect application

In your Halo instance, go to Configuration → Integrations → API and click New. Configure as follows:

  1. Application Name: MCP for AI assistants (anything you like)
  2. Authentication Method: Authorization Code (PKCE)
  3. Allowed Logins: check Allow Agent Logins. Leave the others unchecked.
  4. Redirect URI:
    https://tools.iusehalo.com/auth/callback

    This is the universal callback for every iusehalo tool — Outlook, MCP and any future ones. Register once and never touch this list again.

  5. Permissions / Scopes: all (or the minimum needed — tickets, users, clients, actions).
  6. Save the application, then copy the generated Client ID for the next step.
CORS isn't required for MCP. The MCP server handles token exchange server-to-server, so the Halo Connect CORS whitelist can stay empty unless you also run the Outlook add-in (which needs https://tools.iusehalo.com).

Add it to your assistant

  1. Open claude.ai → Settings → Connectors.
  2. Click Add custom connector.
  3. Paste the URL above as the server URL. Name it Halo PSA (or whatever you like).
  4. Click Connect — you'll be sent to your Halo login. Sign in normally.
  5. Tools will appear under the connector; try asking Claude to "find the Halo contact for <email>".
